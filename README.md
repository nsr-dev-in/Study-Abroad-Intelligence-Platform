# Study Abroad Intelligence Platform

## Business Problem
Students struggle to compare countries based on cost, education quality, visa success, job opportunities and ROI.

## Objective
Build a data-driven decision support platform that helps students choose the best study abroad destination.

## Stakeholders
- Students
- Parents
- Universities
- Study Abroad Consultants

## Key Questions
- Which country offers the best ROI?
- Which country has the highest visa approval rate?
- Which country is most affordable?
- Which country provides the best career opportunities?

## Deliverables
- Clean Dataset
- PostgreSQL Database
- SQL Analysis
- Power BI Dashboard
- Streamlit Web Application
- Recommendation Engine
